## Projeto

O DevRadar é um projeto que visa conectar desenvolvedores próximos a você que trabalham com as mesmas tecnologias.

## Features

Esse projeto foi desenvolvido com as seguintes tecnologias:

- [React](https://reactjs.org)
- [React Native](https://facebook.github.io/react-native/)
- [Node.js](https://nodejs.org/en/)
- [Expo](https://expo.io/)
- [TypeScript](https://www.typescriptlang.org/)
- [Socket IO](https://socket.io/)

## Utilizando o projeto

1. Clone o repositório utilizando `git@github.com:yuriazevedo11/dev-radar.git`
2. Entre na pasta do repositório: `cd dev-radar`
3. Rode `yarn bootstrap` para o instalar todas dependências do projeto

### Iniciando o servidor backend

1. Entre na pasta backend: `cd backend`
2. Rode `yarn dev` para iniciar o servidor

### Iniciando a aplicação web

1. Entre na pasta frontend: `cd frontend`
2. Rode `yarn start` para iniciar a aplicação

### Iniciando a aplicação mobile

1. Entre na pasta mobile: `cd mobile`
2. Caso queira utilizar um emulador rode `yarn ios` ou `yarn android`
3. Caso queira utilizar em seu próprio celular rode `yarn start`, baixe o aplicativo do Expo em seu celular, e escaneie o qrcode

OBS: Caso tenha escolhido rodar pelo emulador Android você terá que abri-lo antes de rodar o comando, no iOS ele abre sozinho. Para escanear em dispositos Android, abra o app do Expo em seu celular na aba Projects. Para escanear em dispositos iOS basta utilizar a câmera clicar no link.
